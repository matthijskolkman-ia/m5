# Weaving Visualizer

A lightweight browser tool for visualizing weaving structures.

## Features

- Interactive warp and weft thread counts
- Pattern selection: plain, twill, basket, herringbone, and diamond draft
- Custom warp and weft colors
- Grid overlay for structure visualization

## Usage

1. Open `index.html` in a browser.
2. Adjust the controls and click `Redraw`.
3. Resize the browser window to refresh the layout automatically.
